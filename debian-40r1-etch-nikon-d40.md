# Debian 4.0r1 (Etch) and Nikon D40

Wikipedia article with some reviews and more information can be found [here](https://en.wikipedia.org/wiki/Nikon_D40).

The **f-spot** didn’t see the camera, and I don’t use Picasa so I decided to do it manually. Simply connected the USB cable, read the `dmesg` output, created a mount point (exactly like in USB memory sticks), and it worked.
